## JavaScript Notes

### Map

```
const myArray = [1,2,3,4]

myArray.map( el => el + 1)
// returns [2,3,4,5]

myArray.map( el => "c")
// returns ["c", "c", "c", "c"]

myArray.map(() => 1 )
// returns [1,1,1,1]
```
